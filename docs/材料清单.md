1. 打开项目目录，展示 `repos/hotel--ota-ai/` 是本地 Git 代码库。
2. 执行 `python3 scripts/build_change_summary.py --repo hotel--ota-ai --requirement "..."`
3. 展示生成的 `outputs/change_summary-latest.md`。
4. 执行 `python3 scripts/apply_requirement_change.py --repo hotel--ota-ai --requirement "..."`
5. 展示 `git diff` 和 `outputs/codex_patch.diff`。
6. 总结：Codex 负责理解和修改代码，V27 负责业务影响翻译，Python 负责流程编排。
