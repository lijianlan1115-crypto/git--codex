## 十一、字段缺口

| 表格要求字段 | 飞猪页面状态 | 处理建议 |
|---|---|---|
| RevPAR | 暂未在飞猪当前页面直接看到 | 由房费收入/可售房量或 PMS 计算 |
| ADR | 飞猪显示间夜单价，可映射为飞猪口径 ADR | 与 PMS ADR 区分口径 |
| CPC/点击均价 | 全网推未展示，直通车待确认 | 从直通车数据中心或报表导出采集 |
| 推广曝光/点击 | 全网推未展示，直通车待确认 | 从直通车数据中心或 API 采集 |
| 图片质量/HOS 历史/入口标签质量 | 页面入口可查，但质量判断仍需人工/视觉模型 | 飞书表单补采 |
| 核心竞对名单 | 同行动态有同行列表/价库对比，但仍需人工确认核心竞对 | 飞书表单维护 |
| 整改动作与复盘原因 | 平台不自动知道业务原因 | 飞书表单/任务中心记录 |
