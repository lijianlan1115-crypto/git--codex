| **核心职责** | 统一管理PMS/OTA/竞对/推广/环境行情/抖音/会员全量数据采集，全覆盖所有指标；管理采集频率、数据质量校验、异常重试；v5新增：核心圈竞对流失数据采集、竞对转化漏斗数据采集、各平台划线价采集、推广余额采集、扫码统计采集 |

| **v5竞品吸纳** | 吸纳超级店长：核心圈竞对流失数据+竞对转化漏斗数据+HOS分历史数据+用户来源数据+图片质量分；吸纳甩手掌柜：各平台划线价+推广余额+扫码统计+周边活动日历 |

#### SK03 销售基准线引擎

| 项目 | 定义 |
|------|------|
| **编号** | SK03 |
| **名称** | 销售基准线引擎 |
| **所属智能体** | 定价智能体 |
| **核心职责** | 基于历史订单+成本结构+环境行情+利润目标+客源结构，生成每日销售基准线；持续追踪完成率；输出周度汇总报告和月度达标率；基准线基于利润测算排表优化；v5新增：客源结构影响基准线侧重点+周边活动日历影响远期基准线 |
| **输入** | ①历史订单数据（近90天）②月运营成本、单房变动成本、房间总数 ③淡旺季周期日历、节日效应日历 ④竞对远期价格 ⑤商圈供需指数 ⑥预订增速 ⑦投资回报周期 ⑧月租金成本、月人工成本 ⑨客源结构数据（商务/旅游占比）（v5新增）⑩周边活动日历（v5新增） |
| **输出** | ①每日销售基准线（分房型、分渠道）②实时完成进度 ③周度汇总报告 ④月度达标率 ⑤利润测算排表 ⑥实时经营指标推送（每小时汇总，v5新增） |
| **优先级** | **P0** |
| **v5竞品吸纳** | 吸纳甩手掌柜：实时经营指标每小时推送（偏差触发+定时汇总双模式）；吸纳超级店长：客源结构影响基准线侧重点；基准线概念是独创，两个竞品都没有 |
