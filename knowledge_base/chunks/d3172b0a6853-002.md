## 十二、代码改动记录

### 2026-06-08 — RS01 入住表接入 S5 预期出租率

**改了什么：**
- `runtime/storage.py`: 新增 `in_house_guests` 表 + `import_rs01()` 方法
- `runtime/adapters/database.py`: 新增 `rs01_expected_occupancy()` 函数（基于离店时间算预期出租率）
- `runtime/decisions/pricing.py`: `revenue_decision()` 优先使用 RS01 预期出租率替代原始实时在住率
- `runtime/cli.py`: 注册 `import-rs01` 命令

**使用方式：**
```bash
python runtime/hotel_ota_runtime.py import-rs01 --path <RS01_Excel文件路径> --hotel-id puyue
python runtime/hotel_ota_runtime.py revenue-decision --hotel-id puyue
```

**效果：**
- RS01 的 离店时间(checkout_time) 字段直接驱动预期出租率
- 离店时间 > 今天 = 续住 → 计入预期
- 所有离店时间 <= 今天 = 今日退房 → 不计入
- 预期出租率 = 续住间数 / 31
