# 飞猪 OTA 平台数据采集字段清单

采集时间：2026-06-10 
渠道：飞猪 Ebooking 
酒店：贵阳璞悦·奢电竞酒店 
用途：对齐《酒店 OTA 全面诊断系统》表格中必须从 OTA 平台获取的数据，用于 S14 诊断、网页报告和历史复盘。

## 采集原则

- 只采集经营诊断所需字段，不保存住客姓名、手机号、完整订单号等个人敏感信息。
- 订单明细如需落库，应先脱敏或聚合为日期、房型、订单状态、间夜、金额等诊断字段。
- 飞猪渠道字段名与美团字段名不同，统一进入系统时应保留 `platform = fliggy` 和原始字段名。
- Safari 可读取大部分页面，但飞猪后台提示推荐 Chrome；后续自动化采集建议使用 Chrome 登录态或飞猪开放 API/报表导出。
